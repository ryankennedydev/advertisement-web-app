# ADS Platform
🇧🇷 Português
## Sobre

ADS Platform é uma aplicação web que permite aos usuários criar e publicar anúncios com imagens, descrições e links externos. O projeto inclui uma interface simples de autenticação, validação de formulários, personalização de tema e renderização dinâmica de conteúdo utilizando JavaScript.

## Funcionalidades

* Interface de autenticação de usuários
* Criação e publicação de anúncios
* Suporte para upload de imagens
* Alternância entre tema claro e escuro
* Validação de formulários
* Notificações de sucesso e erro
* Interface responsiva

## Tecnologias

* HTML5
* CSS3
* JavaScript

## Objetivo

Este projeto foi desenvolvido com o objetivo de aprimorar conhecimentos fundamentais de desenvolvimento front-end, incluindo manipulação do DOM, tratamento de eventos, validação de formulários, geração dinâmica de conteúdo e construção de interfaces responsivas utilizando tecnologias web modernas.

# ADS Platform
🇺🇸 English
## About

ADS Platform is a web application that allows users to create and publish advertisements with images, descriptions, and external links. The project includes a simple authentication interface, form validation, theme customization, and dynamic content rendering using JavaScript.

## Features

* User authentication interface
* Ad creation and publishing
* Image upload support
* Light/Dark mode toggle
* Form validation
* Success and error notifications
* Responsive user interface

## Technologies

* HTML5
* CSS3
* JavaScript

## Purpose

This project was developed to strengthen core front-end development skills, including DOM manipulation, event handling, form validation, dynamic content generation, and responsive design using modern web technologies.
